API 우선순위 및 변환 방법
1
카카오 로컬 API — REST API 키 필요
정확도 최상 · 한국 주소에 최적화 · 월 300,000건 무료
2
Nominatim (OpenStreetMap) — 무료, 키 불필요
정확도 보통 · 초당 1건 제한 · API 키 없을 때 자동 사용
3
수동 입력 — 자동 변환 실패 시
변환 실패 행은 CSV로 별도 다운로드 후 재처리 가능
좌표계 안내
• WGS84 — GPS 표준 좌표계. 카카오맵·구글맵·공공데이터포털 기본값 (권장)
• KATEC — 대한민국 구 표준 (TM 투영, 중부원점). 구형 GIS 시스템 호환용
• UTM-K — GRS80 타원체 기반. 국토지리정보원 표준 (현행)
※ 카카오 API는 WGS84로 반환됩니다. KATEC·UTM-K는 서버 변환이 필요합니다.
